# FanraTech Memory & Visual Identity Guidelines

We have aligned the platform visual direction specifically for **FanraTech**:
- **Exclusive Color Code 1 (Putih/Off-White):** `#F9FAFB`
- **Exclusive Color Code 2 (Hitam/Midnight):** `#111827`
- No other colorful active indicators (orange, amber, yellow) are to be used in any components.
- Brand logo `FanraTech` is configured to use the Google Font **Lobster** (`font-lobster`).
- Fully responsive styling, maintaining perfect layout from smartphone screen widths up to high-definition laptop formats.
- Speak in **Bahasa Indonesia** to the user (Fanra) at all times.
