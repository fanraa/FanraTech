# FanraTech Development Guidelines & Memory Rules

## Core Brand Rules (DO NOT DEVIATE)
1. **The Core Exclusive Palette:**
   - **Primary Light / Background:** `#F9FAFB` (Putih / Clean Off-White)
   - **Primary Dark / Charcoal Text / Elements:** `#111827` (Hitam / Deep Charcoal Gray)
2. **Accent Restrictions:**
   - **NO ORANGE, AMBER, OR YELLOW ACCENTS.** Absolutely avoid injecting colorful orange, yellow, or pink badges/texts. Keep the entire UI elegant, high-contrast, and luxury-monochromatic using `#111827` and `#F9FAFB`.
3. **Brand Typography ("FanraTech"):**
   - The brand name `FanraTech` on the Navbar, footer, and other core logo contexts MUST always be styled using the **Lobster** font (`font-lobster`).
   - "Fanra" portion is styled dark `#111827`, and "Tech" portion is styled subtle gray `#94A3B8` (or vice versa in dark themes) to show clear and legible contrast.
4. **Responsiveness:**
   - Every single element, card, table, and layout step must be fully responsive, scaling gracefully from ultra-wide laptops and desktops to tiny mobile screen sizes.

## Communication Language
- **Language:** Bahasa Indonesia (Indonesian) must be used in all development conversations with the user (called **Fanra**).
